## Key Features
- PDF and TXT document ingestion
- Vector search pipeline with ChromaDB
- FastAPI backend and browser UI
- Low-latency answers with Groq LLM

## Quick Start
```bash
pip install -r requirements.txt
python rag_backend.py
```

Open `rag_frontend.html` in your browser.
